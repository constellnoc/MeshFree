# MeshFree v0.1.0 发布说明

## 1. 版本定位

`v0.1.0` 是 `MeshFree` 的第一个正式 `MVP` 里程碑版本。

这个版本的目标不是做成大型平台，而是先把最核心的三条业务闭环真正跑通，并形成一个：

- 可以运行
- 可以演示
- 可以部署
- 可以继续维护

的稳定阶段版本。

## 2. 本版已包含内容

当前版本已经包含以下核心能力：

- 游客浏览已审核通过的模型资源
- 游客查看模型详情页
- 游客下载已审核通过的 ZIP 文件
- 游客提交新模型投稿
- 单管理员账号登录后台
- 管理员查看投稿列表与详情
- 管理员通过、拒绝、删除投稿

## 3. 本版文档与工程配套

当前仓库已经包含以下文档和工程支撑内容：

- 中文和英文 MVP 需求文档
- 中文和英文 API 设计文档
- 中文和英文部署文档
- 中文服务器维护文档
- 管理员维护脚本
- `Nginx` 与 `PM2` 部署相关示例配置

## 4. 本版明确不包含的内容

为了保持 `MVP` 范围稳定，本版明确不包含以下功能：

- 普通用户注册与登录
- 搜索
- 分类筛选
- 评论
- 收藏
- 点赞
- 3D 在线预览
- 对象存储
- Docker 部署
- 多管理员系统

## 5. 技术栈

- 前端：`React + TypeScript + Vite`
- 后端：`Node.js + Express + TypeScript`
- 数据库：`SQLite`
- ORM：`Prisma`
- 进程管理：`PM2`
- 反向代理：`Nginx`
- 文件存储：服务器本地磁盘

## 6. 当前部署状态

截至 `v0.1.0`：

- 真实服务器部署已经完成
- 主域名已接入现有部署方案
- 生产环境 `CORS` 已收紧
- 管理员登录接口已具备基础限流
- 公开投稿接口已具备基础限流
- 当前基础测试未发现明显问题

## 7. 已知限制

当前版本仍然存在以下限制：

- 上传文件存储在服务器本地磁盘，不是对象存储
- 功能重点仍然是 MVP 闭环，不是完整社区平台
- 当前没有用户系统，因此投稿联系依赖表单中的联系方式字段
- 当前没有搜索、分类、统计等扩展能力

## 9. 后续方向

`v0.1.0` 之后，项目的优先方向将更偏向：

- 继续做线上环境验收
- 记录部署与维护经验
- 如出现问题，优先做小范围修复
- 在版本稳定后，再考虑下一轮可读性、UI 或功能扩展
