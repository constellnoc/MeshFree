# MeshFree v0.3.0-beta.2 发布说明

## 1. 版本定位

`v0.3.0-beta.2` 是 `MeshFree` 当前面向审核流程与标签治理的 beta 版本。

这个版本的重点不是新增一个完全独立的公开展示功能，而是把现有的“上传 -> 审核 -> 公开展示”主流程补得更可操作、更可维护，也更适合本地联调验证。

## 2. 版本摘要

本版主要聚焦：

- 管理员侧标签治理补全
- 首页多标签筛选
- 中文文案统一
- 审核流程可用性提升

一句话：

- 这版的核心是把标签系统从“数据结构已经分层”推进到“管理员界面已经可以实际处理”。

## 3. 本版新增内容

- 新增管理员原始标签治理接口
  - 忽略私有自定义标签
  - 将私有自定义标签绑定到已有公开标签
  - 从私有自定义标签直接创建新公开标签
  - 直接从管理员面板创建公开标签
- 新增管理员页内联表单：
  - 创建新公开标签
  - 绑定原始标签到已有公开标签
- 新增管理员页私有自定义标签可操作卡片
- 新增管理员创建标签时的双语校验字段：
  - `slug`
  - 英文名
  - 中文名
  - `scopeLevel`
- 新增标签治理相关操作的本地化反馈文案

## 4. 本版调整内容

- 将管理员页中的私有自定义标签从只读显示改为可操作治理流程
- 调整管理员页公开标签选择交互，已选标签会单独上浮显示
- 将首页标签筛选从单选改为多选
- 调整多标签筛选后的结果摘要文案
- 继续统一当前 UI 里的中文术语，包括：
  - `Gallery` -> `模型库`
  - `Dashboard` -> `管理面板`
  - `Admin Dashboard` -> `管理员管理面板`
- 为便于本地开发测试，当前本地管理员默认账号调整为：
  - 用户名：`local`
  - 密码：`123`

## 5. 当前标签治理行为

在本版本中：

- 公开模型只绑定受控公开标签
- 用户提交的自定义标签默认保持私有
- 私有自定义标签不会自动变成公开标签
- 管理员可以明确决定一个私有自定义标签：
  - 忽略
  - 归入已有公开标签
  - 创建为新的公开标签
- 新建公开标签时必须同时提供英文名和中文名
- 标签治理动作会直接反映到当前投稿的审核流程中

- 正式的全站中英文切换体验还需要继续打磨
- 预览器打包体积仍然偏大
