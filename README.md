# MeshFree

A lightweight model upload and download website for MVP demonstration.

## Tech Stack

- React
- TypeScript
- Vite
- Node.js
- Express
- Prisma
- SQLite

## Features

- Public model browsing
- Public model download
- Public submission
- Admin review system

## Project Structure

- `client/` frontend
- `server/` backend
- `docs/` project documents

## Status

Project initialization in progress.
